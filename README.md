# 🚀 Power Platform Samples

Bienvenue dans ce dépôt dédié à la **Power Platform** (Power Apps, Power Automate, Power BI et Dataverse).  
L’objectif est simple : **partager des exemples pratiques, des snippets de code et des composants utiles** pour accélérer vos projets.

---

## 📂 Contenu du dépôt
Le dépôt est organisé par grandes catégories :

- **Power Apps**  
  - Formules réutilisables pour Canvas Apps  
  - Composants (msapp) et bonnes pratiques d’UX  
  - Scripts et règles pour Model-driven Apps  

- **Power Automate**  
  - Exemples de flux prêts à l’emploi  
  - Expressions fréquentes pour manipuler vos données  

- **Power BI**  
  - Formules DAX récurrentes  
  - Exemples de rapports et visualisations  

- **Dataverse**  
  - Plugins et exemples de code C# / JS  
  - Appels API et schémas de tables  

---

## 📌 Évolution du dépôt
➡️ Des fichiers et exemples seront ajoutés **progressivement** afin d’enrichir la base.  
➡️ L’objectif est de construire une **bibliothèque vivante** pour la communauté francophone.
